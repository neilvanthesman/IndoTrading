# Indotrading Official Website Remake (React & Vite)
## This project is a remake of the Indotrading official website, developed using React and Vite as part of an internship assessment test.
### Features
- Navigation Bar
- Search Bar
- Login & Register Buttons
- Carousel Section for Banners
- E-commerce Product Display with Categories
- Footer with Logo and Hyperlinks
- Fully Responsive Layout
- Custom CSS Styling

### Installation
`cd indotrade`
`npm install`
`npm run dev`
